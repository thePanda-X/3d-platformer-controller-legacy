# Old Unity 3D Character Controller (2022/23)

This repository contains an older Unity 3D character-controller prototype I built during 2022–2023. It was inspired by platformers such as Super Mario 64 and focuses on movement feel and platforming-style interactions rather than a finished game.

## What's included
- Scenes and example levels: `Assets/Scenes/`
- Controller code and helpers: `Assets/Scripts/`
- Input actions: `InputSystem_Actions.inputactions`
- Prefabs and art assets: `Assets/prefabs/`, `Assets/Materials/`

## How to open
1. Open this folder as a Unity project in the Unity Editor.
2. Open an example scene from `Assets/Scenes/`.
3. Press Play to run the prototype.

## Notes
- The second demo (linked above) shows a more polished prototype that I made but lost and cannot recover — it is not included here.
- This project is a personal prototype from 2022/23 and is not production-ready or actively maintained.
